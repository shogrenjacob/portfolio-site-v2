My Projects

Music Bloggin Web App

This project was birthed from my love of music, as well as my use of apps like reddit and letterboxd. I used Python, HTML/CSS, JavaScript, and SQLite in the Django framework to create my most complex project so far (though more are coming soon!). This app can perform all basic CRUD functions and implements features seen in most social media apps such as post creation, commenting, a star rating for reviews, searching and liking. This app is fully functional just could use a little facelift with the UI.

Weather App

This project is a small web app that produces the weather report for any city asked for. I used HTML/CSS and JavaScript as well as utilized the OpenWeather API to create this app. The weather app got me familiar with handling JSON data and using it to display data to the user in an efficent manner.
